Final Project Proposal

For this project, I want to elaborate on my previous jquery project, which featured a mirror with a moving image which changed when it was clicked on. I simultaneously wanted to develop my introduction "choose your own adventure" page. The goal of the new website is to create a choice based “web browser game” where clicking on different images or links will lead players through various paths, which ultimately tell a story. The mirrors could be thought of as “portals”. Since there are various paths, the player can have a different experience each time. I also am considering embedding music or audio files to create a more immersive experience. I was simultaneously influenced by online args (alternate reality games), specifically due to the mysteries which typically lie within them.
